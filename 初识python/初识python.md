<h2 id="06e6c573">6、python的注释器</h2>


```python
方式1 : 单行注释
     # 注释内容
方式2： 多行注释
    三个单引号： ''' 单行注释''' 
    三个双引号： """多行注释"""
方式3： 再pycharm中
    ctrl +?   注意只能单行注释
    解开注释也是上述ctrl +? 键


python代码的编写规范>>>:PEE8
  1.单行注释如果在代码上方 则#号和内容之间空一格即可
  2.单行注释如果在代码后面，需要线控两格 再#号和内容之间空一格
  3.先自己随便写，用pycharm的快捷键，最后注意修改的地方
     ctrl + alt +l
     详情见图下
```
